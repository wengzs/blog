# 权限

第一个字符：

  1. d：dir，目录
  2. -：文件
  3. l：link，链接
  4. b：接口设备，可随机存取设备
  5. c：串行端口设备，键盘鼠标等一次性读取设备

接下来的字符，三个一组：

  1. 第一组 文件拥有者的权限
  2. 第二组 同群组的权限
  3. 第三组 其他非本群组的权限
  4. r：read
  5. w：write
  6. x：execute
    1. 包括进入目录

# 改变权限

  1. chgrp 改变群组
  2. chown 改变文件拥有者
  3. chmod 改变文件权限
    1. r 4 w 2 x 1

# 限制

  1. 单一文件名长度 255 个字符
  2. 路径 4096 个字符

# 文件与目录管理

  1. 查询 ls
    1. ls -al
  2. 复制 cp
    1. -a = -pdr
    2. -p 复制文件属性
    3. -d 如果为link，则复制为link
    4. -r 递归复制
  3. 删除 rm
    1. rmdir
    2. -r
    3. -f 忽略不存在的文件 不会出现警告信息
  4. 移动 mv
    1. -u 用新的source覆盖dest
    2. 重命名

# 文件内容查看

  1. 直接查看
    1. cat 从第一行开始显示
    2. tac 从最后一行开始显示
    3. nl 带行号
  2. 翻页查看
    1. more
    2. less
  3. 数据截取
    1. head
    2. tail
  4. 修改文件时间与新建
    1. touch
  5. 查看非纯文本文件
    1. od

# 查找命令、文件

  1. which
    1. 在 PATH 中搜索
  2. whereis
    1. 在系统的文件数据库中搜索，可能找到被删除的文件，也可能找不到刚添加的文件。
    2. 数据库默认是在每天更新一次。
    3. 查找二进制文件、源码文件（指令+源码位置）
  3. locate
    1. 在系统的文件数据库中搜索，可能找到被删除的文件，也可能找不到刚添加的文件。
    2. 数据库默认是在每天更新一次。
  4. find
    1. 在磁盘中直接查找
    2. `fin path opt file/dir``

# 权限与命令

  1. 进入目录
    1. 需要 x 权限
    2. 使用ls，还需要 r 权限
  2. 读取文件
    1. cat、more、less 等，对文件需要 r 权限
    2. 对所处目录需要 x 权限
  3. 修改
    1. 使用 vi 需要 r、w 权限
    2. 对所处目录需要 x 权限
  4. 创建
    1. 对所处目录需要 w、x 权限
  5. 运行
    1. 所处目录需要 x 权限
    2. 对目标文件需要 x 权限
