# 个人信息方法论

相较信息本身，获取信息的渠道与能力，更为重要。

## 信息的分类

### 片段

片段信息包含：短信、电话、微信、邮件、App 通知、短视频、流行音乐、体育比赛、新闻等。片段信息维系着我们的社交关系，非常容易占用我们大量的注意力。
从信息输入的渠道数量到阅读它们占用的时间，我们应该做严格的限制。永远不要希望从这些信息中获得能力提升。

### 篇章

篇章是能够自洽的最小信息单元。它能够让你了解某个知识中的一点或几点：`为什么`、`是什么`、`怎么做`。
篇章信息包含：一篇博客、一段教学视频、一部电影、一篇深度报道等。
我们从篇章中获取灵感、结论，而非技能。

### 书卷

书卷能够将某个知识面完整阐述。这个知识面可以是某一技能、规范、定义、原理、方法论等。它的载体有书（纸质或电子）、多个教学视频、完整的在线文档等。
我们能从书卷中获得能力的最小抽象。

### 体系

体系是能力（知识储备、方法论、心态、身体、习惯等）变现（改变现实）的过程产物。我们从目标和理想出发，经历海量的知识储备，在实践的过程中逐步积累经验、方法论和习惯，并且改变了现状，并最终拥有解决同类型问题的的能力体系。
人的一生中，能够形成的能力体系并不多。

## 信息的载体设备

### 手机

手机就像一个闹哄哄的菜市场。手机里的每个 app，都希望占用我们所有的空闲时间，并且尽可能的把钱花在它们上面。手机的屏幕太小，并且没法同生产力设备（生产线、服务器等）打通。手机的唯一价值是处理片段信息。

### pad & kindle

治理得当的 pad 和 kindle 等大屏设备，是阅读的好帮手，前提是我们需要删除这些设备上的“菜市场”应用：新闻、通信、娱乐 App。
大屏设备适合在书卷类信息上做深度阅读、要点标记、知识点复习。

### mac & win

电脑是真正的生产力设备。对于 IT 这个行业，我们能在上面获得绝大部分专业领域的知识体系。

### 纸质书籍

笨重、无法随时获取且越来越不习惯的东西。

## 信息的生命周期

### 目标

目标定义了`为什么`。

### 信息源

根据目标构建信息源，信息源定义了`是什么`。

### 知化

信息知化，即信息经过：`输入` - `解构、记忆、认知` - `实践、总结`三部曲，具象化到个人身上所形成的能力。知化即`怎么做`。
